# SecureTrack – Project Outline (Android App)

## 1. Project Description

SecureTrack is a mobile-based cybersecurity awareness application designed to help users better understand and improve their personal security habits. The application provides a simple and user-friendly interface that allows individuals to assess their current cybersecurity practices and receive a basic risk score along with actionable recommendations. The goal of this application is to simplify complex cybersecurity concepts into something practical and easy for everyday users to understand and apply.

---

## 2. Problem Addressing

Many individuals lack visibility into their personal cybersecurity risk. Users commonly reuse passwords, fail to enable multi-factor authentication, and are often unaware of potential vulnerabilities. This creates significant exposure to security threats such as data breaches and unauthorized access. SecureTrack addresses this problem by providing users with a structured way to evaluate their habits and receive guidance on improving their security posture.

---

## 3. Platform

The application will be developed primarily for the Android platform using Android Studio and the Android SDK. Android was selected due to its widespread use and compatibility with various devices, making it an ideal platform for reaching a broad audience.

---

## 4. Front-End / Back-End Support

### Front-End:
- Android UI components (Activities, layouts, basic navigation)
- User input forms for security-related questions
- Display screens for risk results and recommendations  

### Back-End (Conceptual for now):
- Cloud-based data storage (future implementation)
- Potential API integration for security-related data (future enhancement)
- Local storage for initial prototype functionality  

---

## 5. Functionality

The core functionality of SecureTrack will include:
- User input of security habits (password usage, 2FA, etc.)
- Risk evaluation based on user responses  
- Display a security risk score (low, medium, high)  
- Navigation between screens using Android activities  

---

## 6. Design (Wireframes – Conceptual)

### Screen 1: Welcome Screen
- App title (SecureTrack)  
- "Start Assessment" button  

### Screen 2: Questionnaire Screen
- Series of questions (Yes/No or multiple choice)  
- Input fields for user responses  

### Screen 3: Results Screen
- Display risk score  
- Show recommended actions  
- Option to restart assessment  

### Screen 4 (Optional Future): History / Tracking
- View past results  
- Track improvement over time
